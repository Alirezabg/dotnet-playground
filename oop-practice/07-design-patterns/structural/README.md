# Structural Patterns

Focus: **Decorator** and **Adapter** in the Product Catalogue context.

## Decorator Task
Add caching behaviour to `IProductRepository` using a `CachingProductRepository` decorator —
without modifying the original repository.

## Adapter Task
You have a third-party `LegacyInventorySystem` with its own API.
Write an adapter that makes it conform to your `IInventoryRepository` interface.
